concat()

Description:

The concat() method is used to join two or more strings/ arrays.This will take 1 or more values as parameters.

Syntax:

arr1.concat(arr2, arr3, ..., arrn)

splice()

Description:

The splice() method adds/removes items to/from an array.

Syntax:

array.splice(number1,number2,value1,value2,....., valuen)

number1---An integer that specifies at what position to add/remove elements.

number2---The number of elements to be removed. If we give 0, no element will be removed.

value1,value2,.....valuen---are n elements to be added to array.

If we don't give any values after 2nd second number,then the specified number of values are deleted but no new values are added.

Example:

If a=["Hari",Keerthan","Pritham","Rishi","Ruthvik"]

a.splice(0,2,"Yokith","Sree") gives

Explanation:

My 1st number is 0 and 2nd number is 2 i.e.,starting from 0 index,2 elements are deleted and "Yokith","Sree" are added in 1st and 2nd index values.
